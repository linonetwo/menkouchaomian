# 门口炒面
「只要炒面大叔开心了，炒出来的东西应该会更好吃吧」

## usage

1.
  输入用户名，方便大叔知道炒出来的面是谁的。
  此时输入炒面大叔专属用户名可进入后台。
  为保证流畅性，免密码登陆。

2.
  往菜单加入配料。

3.
  点选已有的配料，配好后提交订单，炒面大叔将会看到订单，你将会看到排队时间。

4.
  炒面大叔炒好后，你会收到通知叫你下去拿面。
